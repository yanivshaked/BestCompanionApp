# BestCompanion Website

Official website for the BestCompanion Android app — an always-on calendar display designed for people living with dementia, Alzheimer's disease, and age-related memory challenges.

## Pages

- **index.html** — Landing page with features, use cases, and download links
- **privacy-policy.html** — Privacy policy for the app

## Features

- Auto-detects browser language
- Manual language selector (20 languages supported)
- Right-to-left support for Arabic and Hebrew
- Fully responsive design
- Dark theme matching the app's visual identity

## Hosting

This site is hosted via GitHub Pages. To deploy, enable GitHub Pages in the repository settings pointing to the root of the `main` branch.
